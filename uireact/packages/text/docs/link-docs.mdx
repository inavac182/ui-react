# UiLink

<sup>
  <a href="https://github.com/inavac182/ui-react/blob/main/packages/text/" target="_blank">
    v{packageJson.version}
  </a>
</sup>

> Component to render a wrapper around links to provide styling, you have to provide the link element, this was done so we could support
> multiple use cases Link from react router, or Link from next JS or any other router you are using.

## Installation

> npm i @uireact/foundation @uireact/text

## UiLink

<Playground>
  <UiLink>
    <a href="https://uireact.io/">Some text</a>
  </UiLink>
</Playground>

## UiLink with color category

<Playground>
  <UiLink category="error">
    <a href="https://uireact.io/">Some text</a>
  </UiLink>{' '}
  <UiLink category="positive">
    <a href="https://uireact.io/">Some text</a>
  </UiLink>{' '}
  <UiLink handleClick={() => console.log('Link Clicked')} category="negative" fontStyle="italic">
    <a href="https://uireact.io/">Some text</a>
  </UiLink>
</Playground>
