# @UiReact/create-next-app

This script pulls the [Next JS project](https://github.com/inavac182/next-js-w-uireact) with [@UiReact](https://github.com/inavac182/uireact) all up and running.

## Usage

1. Run script

```
npx @uireact/create-next-app
```

2. Start it up

```
npm run dev
```
