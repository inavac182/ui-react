# UiBadge

> Component mainly used to show badges to categorize information

## Installation

> npm i @uireact/foundation @uireact/badge

## UiBadge

```jsx live scope={{ UiSpacing, UiBadge, UiIcon }}
  <>
    <UiSpacing margin={{ block: 'three' }}>
      <UiBadge size="xlarge">✈️ Primary</UiBadge>
    </UiSpacing>
    <UiSpacing margin={{ block: 'three' }}>
      <UiBadge category="secondary" size="large">
        <UiIcon icon="BadgeCheck" /> Secondary with icon
      </UiBadge>
    </UiSpacing>
    <UiSpacing margin={{ block: 'three' }}>
      <UiBadge category="tertiary" size="regular">
        🚀 Tertiary
      </UiBadge>
    </UiSpacing>
    <UiSpacing margin={{ block: 'three' }}>
      <UiBadge category="positive">✅ Positive</UiBadge>
    </UiSpacing>
    <UiSpacing margin={{ block: 'three' }}>
      <UiBadge category="warning" size="xsmall">
        🚨 Warning
      </UiBadge>
    </UiSpacing>
  </>
```
