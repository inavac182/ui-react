# UiDigitsInput
<Metadata packageName='form' packageJson={packageJson} />

> Component used to render digits inputs

## Installation

> npm i @uireact/foundation @uireact/form

## Usage

```jsx live scope={{UiDigitsInput}}
  <UiDigitsInput
    label="Introduce your code"
    digits={6}
    onChange={(value) => {
      console.log(value);
    }}
    onComplete={(code) => {
      console.log(`completed ${code}`);
    }}
  />
```

### UiDigitsInput with size and category

```jsx live scope={{UiDigitsInput}}
  <UiDigitsInput
    label="Introduce your code"
    digits={6}
    onChange={(value) => {
      console.log(value);
    }}
    onComplete={(code) => {
      console.log(`completed ${code}`);
    }}
    size="xlarge"
    category="error"
    error="Code is not valid"
  />
```
