## Installation

> npm i @uireact/foundation @uireact/charts

## UiLinearChart

This component will take 2 main values: `currentValue` and `limitValue`, the `limitValue` becomes the 100% and the current value will get the bar to its corresponding percentage, if the current value is bigger than the limit then it
will default to 100%.

<Playground>
  <UiLinearChart title="Demo" currentValue={10} currentLabel="start value" limitValue={100} limitLabel="End value" />
</Playground>
