> This chart receives 2 values, the limit (100% of the chart) and a current which will calculate the percentage and represent it in the linear chart.

## Installation

> npm i @uireact/foundation @uireact/charts

## UiLinearChart

This component takes 2 values: `currentValue` and `limitValue`, the `limitValue` is the 100% of the chart and the current value represents how much of the `limitValue` is covered, if the current value is bigger than the limit then it
will default to 100%.

<Playground>
  <UiLinearChart title="Demo" currentValue={15} limitValue={30} />
</Playground>

## UiLinearChart - with labels

The limit label appears at the top and right edge of the chart, the current value renders in the bottom and moves within the current edge line.

<Playground>
  <UiLinearChart title="Demo" currentValue={40} currentLabel="$40" limitValue={100} limitLabel="$100" />
</Playground>

## UiLinearChart - with static label position

When static label is used the label for the current value will be static at the bottom of the chart

<Playground>
  <UiLinearChart
    currentValue={300}
    currentLabelPosition="static"
    currentLabel="300 lb"
    limitValue={1000}
    limitLabel="1000 lbs"
  />
</Playground>

## UiLinearChart - with custom colors

The custom colors supports hex and rgb colors

<Playground>
  <UiLinearChart
    title="Demo with color"
    currentValue={60}
    currentColor="rgb( 28, 225, 127)"
    limitValue={80}
    limitColor="#BE3708"
  />
</Playground>

## UiLinearChart - with custom colors per theme

When using the `ThemedColor` for the color you can set up a different color per theme per each value

<Playground>
  <UiLinearChart
    title="Demo with color"
    currentValue={60}
    currentLabel="60 Mb used"
    currentColor={{ dark: '#19BE08', light: '#6E45EE' }}
    limitValue={80}
    limitLabel="80 mbs available"
    limitColor={{ dark: '#C53CE3', light: '#45EEDF' }}
  />
</Playground>
