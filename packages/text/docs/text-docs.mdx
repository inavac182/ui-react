## Installation

> npm i @uireact/foundation @uireact/text

## UiText

<Playground>
  <UiText>Some text</UiText>
</Playground>

By default the text is rendered inside a `p` tag

## UiText inline

<Playground>
  <UiText inline>Some text</UiText>
</Playground>

When using inline the text will be rendered in a `Span` tag
