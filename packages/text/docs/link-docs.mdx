# UiLink

<sup>
  <a href="https://github.com/inavac182/ui-react/blob/main/packages/text/" target="_blank">
    v{packageJson.version}
  </a>
</sup>

> Component to render a wrapper around links to provide styling, depending on our use case we would render an anchor,
> or Link from react or Next or any other framework as at the end they will render an anchor to the DOM to which the style is attached.

## Installation

> npm i @uireact/foundation @uireact/text
