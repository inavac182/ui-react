## Installation

> npm i @uireact/foundation @uireact/grid

## Usage

### 1. Using just UiGrid

<Playground>
    <UiGrid cols={3} rows={2} colsGap='10%' justifyItems='center'>
        <UiCard>Item 1!</UiCard>
        <UiCard>Item 2!</UiCard>
    </UiGrid>
</Playground>

> This is the suggested approach if all grid items will share the same styles.


### 2. Using UiGrid with UiGridItem

<Playground>
    <UiGrid cols={3} rows={2} colsGap='5px'>
        <UiViewport criteria='l|s'>
            <UiGridItem cols={2}>
                <UiCard>Item 1!</UiCard>
            </UiGridItem>
        </UiViewport>
        <UiGridItem palceSelft='stretch' rows={2}>
            <UiCard>Item 2!</UiCard>
            <span>Yeah, 2 rows is nice!</span>
        </UiGridItem>
    </UiGrid>
</Playground>

> In this case as we are using the UigridItem we are able to manipulate each grid item on its own.

> Also notice that in Item 1 we are making use of the [UiViewport](./packages-foundation-docs-viewport) to render/remove it depending on the breakpoint
