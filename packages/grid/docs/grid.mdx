> Structure and organize the UI using columns

## Installation

> npm i @uireact/foundation @uireact/grid

## Usage

### 1. Using just UiGrid

<Playground>
  <UiGrid cols={3} rows={2} colsGap="10%" justifyItems="center">
    <UiCard>
      <UiText>Item 1!</UiText>
    </UiCard>
    <UiCard>
      <UiText>Item 2!</UiText>
    </UiCard>
  </UiGrid>
</Playground>

> This is the suggested approach if all grid items will share the same styles.

### 2. Using UiGrid with UiGridItem

<Playground>
  <UiGrid cols={3} rows={2} colsGap="5px">
    <UiViewport criteria="l|s">
      <UiGridItem cols={2}>
        <UiCard>
          <UiText>Item 1!</UiText>
        </UiCard>
      </UiGridItem>
    </UiViewport>
    <UiGridItem placeSelf="stretch" rows={2}>
      <UiCard>
        <UiText>Item 2!</UiText>
        <UiText>Yeah, 2 rows is nice!</UiText>
      </UiCard>
    </UiGridItem>
  </UiGrid>
</Playground>

> In this case as we are using the UigridItem we are able to manipulate each grid item on its own.

> Also notice that in Item 1 we are making use of the [UiViewport](./packages-foundation-docs-viewport) to render/remove it depending on the breakpoint

### 3. Using GridBreakpointsDistribution prop

This prop give you the ability to set a number of cols/rows depending on the breakpoint.

<Playground>
  <UiGrid cols={{ small: 1, medium: 2, large: 3, xlarge: 3 }} rows={2} colsGap={20} rowsGap={10}>
    <UiCard>
      <UiText>Item 1!</UiText>
    </UiCard>
    <UiCard>
      <UiText>Item 2!</UiText>
    </UiCard>
    <UiCard>
      <UiText>Item 3!</UiText>
    </UiCard>
  </UiGrid>
</Playground>

> This is the suggested approach if all grid items will share the same styles.

### 2. Using custom UiGridItem col span

<Playground>
  <UiGrid cols={4} colsGap="5px">
    <UiGridItem cols={2}>
      <UiCard>
        <UiText>Item 1!</UiText>
      </UiCard>
    </UiGridItem>
    <UiGridItem placeSelf="stretch" cols={2} startingCol={3} startingRow={2}>
      <UiCard>
        <UiText>Item starting on the 3rd col and 2nd row</UiText>
      </UiCard>
    </UiGridItem>
  </UiGrid>
</Playground>
