## Usage

  <UiSpacing margin={{ all: Sizing.five }}>Nice margin dude!</UiSpacing>
</Playground>

**UiSpacing** renders a div to which it assigns its styling and wraps the children

### SizingDistribution

The `SizingDistribution` definition is:

```
type SizingDistribution = {
  /** Applies size to all sides */
  all?: Sizing;
  /** Applies size to top and bottom */
  block?: Sizing;
  /** Applies size to left and right */
  inline?: Sizing;
  /** Applies size to left */
  left?: Sizing;
  /** Applies size to right */
  right?: Sizing;
  /** Applies size to top */
  top?: Sizing;
  /** Applies size to bottom */
  bottom?: Sizing;
};
```

Options `all`, `block` and `inline` are used independently **can't be combined**.
