## Details ü§ì

More information about the `<UiSpacing />` implementation.

### SpacingDistribution

The [SpacingDistribution](https://github.com/inavac182/uireact/blob/main/packages/foundation/src/spacing/spacing.tsx#L8) let you set up spacing
in multiple ways:

```
type SpacingDistribution = {

  all?: Sizing;
  /** Apply spacing to all sides */

  block?: Sizing;
  /** Apply spacing to top and bottom */

  inline?: Sizing;
  /** Apply spacing to left and right */

  left?: Sizing;
  /** Apply spacing to left */

  right?: Sizing;
  /** Apply spacing to right */

  top?: Sizing;
  /** Apply spacing to top */

  bottom?: Sizing;
  /** Apply spacing to bottom */
};
```

Options `all`, `block` and `inline` are used independently **can't be combined**. There is already an issue opened for this to change this.

### Theme

The [Sizing enum](https://github.com/inavac182/uireact/blob/main/packages/foundation/src/types/enums/spacing.ts#L1) is used to reference a
value from the [Spacing](https://uireact.io/packages-foundation-docs-themes#spacing-) property provided from the themes

For instance if you set up this in your theme:

```
{
  ...,
  spacing: {
    one: '1px',
    two: '2px',
    three: '4px',
    four: '8px',
    five: '16px',
    six: '32px',
    seven: '64px',
  },
  ...
}
```

If you use this:

```
<UiSpacing padding={{all: Sizing.three}}>
  Some content
</UiSpacing>
```

The content will get wrapped under a `div` element with the CSS value of: `padding: 4px;`
