# Themes

@uireact provides with a default theme that you can use:

## Light coloration

[Light Theme](https://github.com/inavac182/uireact/blob/main/packages/foundation/src/themes/default-colors/light.ts)

## Dark coloration

[Dark Theme](https://github.com/inavac182/uireact/blob/main/packages/foundation/src/themes/default-colors/dark.ts)

<ThemeTokens colors={DefaultTheme.dark} />

## Structure

Each theme has to meet the structure of the [Theme](https://github.com/inavac182/uireact/blob/main/packages/foundation/src/types/themes/theme.ts) type.

The color tokens should go from lighter (token_10) to darker (token_200).
