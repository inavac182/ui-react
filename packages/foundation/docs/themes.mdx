# Themes

@uireact provides with 2 default themes:

## Light

[Light Theme](https://github.com/inavac182/uireact/blob/main/packages/foundation/src/themes/light.ts)

<ThemeTokens colors={DefaultTheme.light} />

## Dark

[Dark Theme](https://github.com/inavac182/uireact/blob/main/packages/foundation/src/themes/dark.ts)

<ThemeTokens colors={DefaultTheme.dark} />

## Structure

Each theme has to meet the structure of the [Theme](https://github.com/inavac182/uireact/blob/main/packages/foundation/src/types/themes/theme.ts) type.

The color tokens should go from lighter (token_10) to darker (token_200).
