# UiExpandoText

<sup>
  <a href="https://github.com/inavac182/ui-react/blob/main/packages/expando/" target="_blank">
    v{packageJson.version}
  </a>
</sup>

> This component renders a text and an icon that when expanded reveals some content underneath it.

## Installation

> npm i @uireact/foundation @uireact/icons @uireact/expando

## Just a expando

<Playground>
  <UiExpandoText label="Expand me">
    <span>This is some content</span>
  </UiExpandoText>
</Playground>

## Expando with custom category and size

<Playground>
  <UiExpandoText label="Expand me large" category="positive" size="xlarge">
    <span>This is some content</span>
  </UiExpandoText>
</Playground>

## Expando initiated expanded

<Playground>
  <UiExpandoText label="Expand me" expanded>
    <span>This is some content</span>
  </UiExpandoText>
</Playground>
