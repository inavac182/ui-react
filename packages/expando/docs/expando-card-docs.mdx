# UiExpandoCard

<sup>
  <a href="https://github.com/inavac182/ui-react/blob/main/packages/expando/" target="_blank">
    v{packageJson.version}
  </a>
</sup>

> This component renders a text and an icon that when expanded reveals some content underneath it.

## Installation

> npm i @uireact/foundation @uireact/icons @uireact/expando

## Just a expando

<Playground>
  <UiExpandoCard expandLabel="Expand me" collapseLabel="Collapse me">
    <span>This is some content</span>
  </UiExpandoCard>
</Playground>

## Expando with custom category and inverse heading coloration

<Playground>
  <UiExpandoCard expandLabel="Expand me" collapseLabel="Collapse me" category="tertiary" headingInverseColoration>
    <span>This is some content</span>
  </UiExpandoCard>
</Playground>

## Expando initiated expanded

<Playground>
  <UiExpandoCard expandLabel="Expand me" collapseLabel="Collapse me" expanded>
    <span>This is some content</span>
  </UiExpandoCard>
</Playground>
