# UiTable

<sup>

> Used to render information in a table that is filterable and sortable.
> The rows can be selectable if the `onSelect` CB is provided.

## Installation

> npm i @uireact/foundation @uireact/table

## UiTable

<Playground>
  <UiTable
    data={{
      headings: ['No', 'Summary', 'Price'],
      items: [
        { id: '1', cols: ['1', 'item 1', '$10'] },
        { id: '2', cols: ['2', 'item 2', '$20'] },
        { id: '3', cols: ['3', 'item 3', '$30'] },
        { id: '4', cols: ['4', 'item 4', '$40'] },
        { id: '5', cols: ['5', 'item 5', '$50'] },
        { id: '6', cols: ['6', 'item 6', '$60'] },
        { id: '7', cols: ['7', 'item 7', '$70'] },
        { id: '8', cols: ['8', 'item 8', '$80'] },
        { id: '9', cols: ['9', 'item 9', '$90'] },
        { id: '10', cols: ['10', 'item 10', '$100'] },
      ],
    }}
    onSelect={(id) => {
      if (id) {
        alert(`Row selected ${id}`);
      } else {
        alert('Row unselected');
      }
    }}
  />
</Playground>

- If a row is selected the `onSelect` CB is executed
- If the same row is selected again then its unselected and the `onSelect` CB is executed with empty id.
- If the filter box is used AND there is a row selected, then the state is reset and the `onSelect` CB is executed with empty id.
