# UiProgressIndicator

> Used to represent the progress in a multi step process

## Installation

> npm i @uireact/foundation @uireact/progress-indicator

## UiProgressIndicator

```jsx live scope={{UiProgressIndicator, UiProgressIndicatorItem, UiText}}
<>
  <UiProgressIndicator>
    <UiProgressIndicatorItem>
      <UiText>Step 1</UiText>
    </UiProgressIndicatorItem>
    <UiProgressIndicatorItem>
      <UiText>Step 2</UiText>
    </UiProgressIndicatorItem>
    <UiProgressIndicatorItem>
      <UiText>Step 3</UiText>
    </UiProgressIndicatorItem>
  </UiProgressIndicator>
</>
```
