import { UiHeader } from '@uireact/header';
import { UiSpacing, Sizing } from '@uireact/foundation';

# UiHeader

> Header component, usually use to render logo and a few menu user actions

## Installation

> npm i @uireact/foundation @uireact/button

## Header

The header component will render at the top but on scrolling the header will move with the scrolling

```jsx live scope={{UiHeader, UiSpacing}}
  <div style={{ position: 'relative' }}>
    <div style={{ overflow: 'auto', maxHeight: '300px', backgroundColor: 'lightGray' }}>
      <UiHeader>
        <UiSpacing margin={{ left: Sizing.five }}>
          <p>Header</p>
        </UiSpacing>
      </UiHeader>
      <div style={{ color: 'black' }}>
        <p>Some content</p>
        <p>Some content</p>
        <p>Some content</p>
        <p>Some content</p>
        <p>Some content</p>
        <p>Some content</p>
        <p>Some content</p>
        <p>Some content</p>
        <p>Some content</p>
        <p>Some content</p>
      </div>
    </div>
  </div>
```

## Header fixed

The header component will kept positioned at the top of the content

```jsx live scope={{UiHeader, UiSpacing}}
  <div style={{ position: 'relative' }}>
    <div style={{ overflow: 'auto', maxHeight: '300px', backgroundColor: 'lightGray' }}>
      <UiHeader fixed>
        <UiSpacing margin={{ left: Sizing.five }}>
          <p>Header</p>
        </UiSpacing>
```
